Q1. 英語と数学のテストを75名に行ったところ、以下の結果となった。  
1. 英語の合格点を取った人が35名  
2. 数学の合格点を取った人が20名  
3. 英語、数学どちらも不合格だった人が30名  

テストを受けた人のうち、英語、数学どちらも合格点を取った人は何名か

A1.【回答：10名】  
【概要：75-30=45が少なくともどちらかの科目で合格した人。
数学の合格点のみを取った人は45-35=10
英語の合格点のみを取った人は45-20=25
45-10-25=10がどちらの科目も合格した人。】
